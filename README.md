# choiceJs

## why use choiceJs?

- `if...else` is to large to maintain
- no callback hell
```
if (...) {
  if (...) {
    if (...) {
      if (...) {

      }
    }
  }
} else {
  if (...) {
    if (...) {
      if (...) {

      }
    }
  }
}
``` 

Api:

- add: `function`
- use: `function`
- destroy: `function`
